# IGCSE Learning Hub (IGCSE-LH)

## Overview
IGCSE Learning Hub is a personalized Learning Management System (LMS) designed
for IGCSE Mathematics students. The system provides adaptive learning paths,
AI-assisted marking, and transparent progress tracking for students, teachers,
and parents.

## Tech Stack
- Frontend: ReactJS / Next.js
- Backend: Python (FastAPI)
- Database: MySQL
- AI: OpenAI API
- Version Control: Git & GitHub
